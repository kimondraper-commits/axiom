## Session Rules (Standing Instructions)

- Enter plan mode for ANY non-trivial task (3+ steps or architectural decisions)
- Use subagents to keep main context clean — one task per subagent
- After any correction: update this file with the pattern immediately
- Never mark a task complete without proving it works
- For non-trivial changes: ask "is there a more elegant solution?"
- Bug reports: fix autonomously — no hand-holding, no context switching required
